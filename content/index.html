The inclusion above is an autogenerated list of blog posts (see content/blog/mkblog.sh, which is invoked by make). When including files using {{##filename##}}, cyc looks for filename in

  1. … the folder of the source file,
  2. … template/,
  3. … static/,
  4. … in the present working directory.

The latter allows you include any file using absolute paths. For example, the hostname of the device that generated this page was {{##/etc/hostname##}}.

In this example setup, the "Last updated" field (normally stored in file.html.mtime) can be overriden by writing to file.html.utime. See the current page for an example ({{!!mtime!!}}). pre.sh will then use file.html.utime. Of course, it is not at all mandatory to use the "Last updated" field. Feel free to just remove {{!!mtime!!}} from your templates.
